# 아이캔스토리 (I CAN STORY) 웹사이트

이 프로젝트는 아이캔스토리의 공식 웹사이트 MVP 버전입니다. Next.js 14, Tailwind CSS, TypeScript로 제작되었습니다.

## 🚀 배포 방법 (Vercel 권장)

이 프로젝트는 [Vercel](https://vercel.com/)에 최적화되어 있습니다.

1. **GitHub에 코드 푸시**
   - 이 프로젝트 폴더를 GitHub 저장소에 푸시합니다.

2. **Vercel 프로젝트 생성**
   - Vercel 대시보드에서 `Add New...` > `Project`를 클릭합니다.
   - GitHub 저장소를 불러옵니다.
   - `Framework Preset`이 `Next.js`로 자동 설정되었는지 확인합니다.
   - `Deploy` 버튼을 클릭합니다.

3. **완료!**
   - 배포가 완료되면 제공된 URL로 웹사이트에 접속할 수 있습니다.

## 🛠 로컬 개발 환경 설정

로컬에서 실행하려면 Node.js (v18 이상)가 필요합니다.

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

## 📁 주요 기능

- **홈 (`/`)**: 서비스 소개 및 주요 타겟 네비게이션
- **숏폼 제작 (`/shortform`)**: 서비스 상세 및 가격 정책
- **마케팅 교육 (`/education`)**: 커리큘럼 소개
- **성공 사례 (`/cases`)**: 포트폴리오 갤러리 (필터링 기능)
- **문의하기 (`/contact`)**: 상담 문의 폼
- **관리자 (`/admin`)**: 콘텐츠 관리 (데모 버전)

## ⚠️ 주의사항

- **데이터 저장**: 현재 버전(MVP)에서는 데이터가 로컬 파일(`data/cases.json`)에 저장됩니다. Vercel 등 서버리스 환경에 배포 시, 새로 등록한 데이터는 영구 저장되지 않고 초기화될 수 있습니다. 정식 서비스 시에는 데이터베이스(Supabase, Firebase 등) 연동이 필요합니다.
- **이미지**: 데모용 이미지는 현재 포함되어 있지 않으므로, 실제 배포 시 `public/images` 폴더에 필요한 이미지를 추가해야 합니다.
